Yelp Camp App

An experimental React front end app that combines with a Mongo/Express/Node back end server to create a website for creating, editing, and reviewing campgrounds from all over the United States.

This app integrates:
- CSS Loader
- Foundation Sites
- Moment
- jQuery
- Node Sass
- React
- React Google Maps
- React Modal
- React-Redux
- Redux-Thunk
- Sass Loader
- Webpack
